- **Tecnología Base:** React con componentes funcionales y Hooks.
    
- **Enrutamiento:** `react-router-dom` para rutas anidadas y protegidas.
    
- **Prohibición de Backend:** No se permite el uso de APIs externas, bases de datos en la nube (Firebase, Supabase) o servidores propios.
    
- **Librerías Permitidas:** Dexie.js para la gestión de IndexedDB.
