**Como** administrador, **quiero** supervisar las reservas de los clientes y tener la opción de cancelarlas **para** gestionar incidencias o cambios de agenda.

- **Criterios de Aceptación:**
    
    - El sistema debe mostrar un listado con el turno asociado, los datos del cliente (nombre y CI) y la fecha de la reserva.
        
    - Debe existir una opción clara para cancelar una reserva, cambiando su estado a "cancelada" en la base de datos.
        
    - Al cancelar una reserva, el cupo disponible del turno asociado debe actualizarse automáticamente.
